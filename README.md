# HR Analytics Dashboard | Power BI

## Overview
This project analyses employee attrition, engagement, job satisfaction, manager satisfaction, workforce composition, and departmental trends using Power BI.

## How to Use
1. Download the `.pbix` file.
2. Open it in Power BI Desktop.
3. Interact with the slicers to filter by department, gender, job level, attrition, and age group.
4. Or, review screenshots in this repository for a quick visual summary.

## Business Objective
The goal of this dashboard is to identify workforce risks and engagement patterns across departments, job levels, gender, and age groups.

## Tools Used

## Key Insights
Sales has the highest attrition rate at 21%.
Human Resources has the most senior-heavy structure.
No department meets the target engagement score.
Exited HR staff show the highest job dissatisfaction.
R&D shows relatively stronger manager satisfaction than other departments.
